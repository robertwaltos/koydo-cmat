# koydo-cmat

CMAT exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM040 |
| Name | CMAT |
| Country | India |
| Category | Admissions |
| Tier | Tier 3 |
| Wave | Wave 5 |

## Legal

This app is a study aid based on publicly available syllabus and exam patterns. Not affiliated with any official examination authority.
